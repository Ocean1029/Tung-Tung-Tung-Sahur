# Town Pass

台北通 App - 集章系統專案

## 📁 專案結構

```
TownPass/
├── frontend/          # Flutter 前端應用
│   ├── lib/          # Dart 程式碼
│   ├── android/      # Android 平台配置
│   ├── ios/          # iOS 平台配置
│   └── ...
│
├── backend/          # 後端 API 服務
│   ├── src/         # 後端程式碼
│   └── ...
│
└── docs/             # 專案文件
    ├── API.md       # API 規格文件
    ├── ARCHITECTURE.md  # 架構說明
    └── CONTRIBUTING.md  # 協作指南
```

## 🛠️ 技術棧

- **前端**: Flutter + Dart
- **後端**: (待定)
- **狀態管理**: GetX

## 🚀 快速開始

### 前端開發

```bash
cd frontend

# 安裝依賴
flutter pub get

# 運行應用
flutter run
```

詳見 `backend/README.md`
